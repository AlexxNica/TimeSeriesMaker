Time Series Maker
  • Github

Do you have an idea of how your data should look like but don't know how to generate it? Is the time series you are thinking about in your mind and you just need to get it in electronic format so that you can start using it in your application or algorithms?

The Time Series Maker allows you to draw a time series and download the result as a CSV or JSON file, pretty neat right?

Export CSV Export JSON

JSON format

The format of the data when exported in JSON format is the following 

{
 'times': [0.0, 0.1, ...],
 'values': [12.5, 13.4, ...]
}

CSV format

The format of the data when exported in CSV format is the following 

time,value
0.0,12.5
0.1,13.4
...
